Downloading and Installing AutoHotkey

Learn how to install AutoHotkey on your computer

1

Visit the AutoHotkey homepage at https://autohotkey.com.


2

Click the download button and download the current version.


3

Run the installation wizard and click through it until you reach the install button.


Choose Unicode over ANSI if prompted.

4

Click install.
